# Khalzam

## About
This is a port of my recent project
[khalzam-go](https://github.com/kisasexypantera94/khalzam).

## Setup
You need to create and initialize database:
```
$ cd src/db
$ createdb -O user databasename
$ psql -f initdb.sql databasename
```
